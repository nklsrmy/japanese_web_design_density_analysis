# japanese_web_design_density_analysis
Machine learning-based analysis of Japanese website information density, UX structure, and adaptation to modern web design trends.

**Objectives**
- Quantify information density in Japanese websites
- Compare Japanese and Western web design structures
- Identify clusters of traditional and modern design adaptation
- Visualize structural design patterns using Python notebooks

**Methodology**
1. Dataset collection from 50 Japanese websites and 25 Western comparison websites
2. Feature extraction using Python scraping tools
3. Data preprocessing and scaling
4. K-Means clustering for pattern discovery
5. PCA-based visualization for cluster interpretation
